# Northstar Console

Northstar Console is a small React and Vite dashboard used as the verified RepoLens preview fixture. It models a realistic frontend with navigation, reusable UI components, a settings workflow, and an API-like service module.

## Pages

- `/` — workspace overview and deploy interaction
- `/settings` — notification and digest preferences

This project is bundled with RepoLens and reviewed before execution. Its package scripts are descriptive only; RepoLens starts it through a fixed Vite CLI command and never executes scripts supplied by a repository.
