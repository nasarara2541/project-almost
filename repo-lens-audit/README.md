# RepoLens audit product brief

RepoLens helps maintainers and contributors answer three questions about a
public repository:

1. What deserves attention?
2. What repository evidence supports that conclusion?
3. What useful contribution can someone make next?

The product performs a bounded, read-only static audit. It never executes the
target repository and does not require a model for its core findings.

See the root `README.md` for setup and verification.
