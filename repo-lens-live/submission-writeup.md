# RepoLens: Submission Write-Up

RepoLens is a live-preview and code-understanding tool for unfamiliar frontend repositories. A developer pastes a public GitHub URL, opens the application in a temporary isolated preview, and explores the routes, components, architecture, and source locations behind what they see.

## Demo narrative

1. Paste the sample repository URL.
2. Show the preview starting.
3. Open the live application.
4. Select a route or visible feature.
5. Show the related component and source files.
6. Ask a natural-language question about the feature.
7. Show the cited trace and highlighted architecture path.
8. Explain how Codex accelerated implementation and how GPT-5.6 powers grounded reasoning.
